# RotateLot
This mod adds a command/cheat to rotate the active build/buy lot.\
\
While in the build mode or the buy mode, press **Ctrl**, **Shift**, and **C** together to open the command console.\
\
For square lots, use *rotateLot+* and *rotateLot-*.\
For non-square lots, you can also use *rotateLot*.
